# DaVinciOS Design Skills Pack

Use this folder as persistent context for your coding extension. It teaches the agent how to design HomeU / DaVinciOS frontend, login screens, and admin backend without copying templates blindly.

Primary design sources:
- shadcn/ui: clean component foundation, login blocks, sidebar/dashboard blocks.
- Magic UI: tasteful animated marketing components.
- Aceternity UI: premium hero sections, backgrounds, cards, motion effects.
- HomeU Admin panel: custom views, authentication, and content management.

Rule: study patterns, do not copy full commercial templates. Rebuild original HomeU-branded components.
