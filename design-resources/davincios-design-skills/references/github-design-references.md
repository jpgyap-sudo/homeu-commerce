# GitHub / UI Reference Map

## shadcn/ui
Use for the core design system.
Reference:
- https://github.com/shadcn-ui/ui
- https://ui.shadcn.com/blocks/login
- https://ui.shadcn.com/blocks/sidebar
- https://ui.shadcn.com/examples/dashboard

Best for:
- Login forms
- Sidebar layouts
- Dashboard pages
- Tables
- Forms
- Command menus

## Magic UI
Reference:
- https://github.com/magicuidesign/magicui
- https://magicui.design

Best for:
- Premium animated sections
- Marketing pages
- Subtle visual polish
- Hero effects

Use carefully in admin backend. Admin should be fast and clear.

## Aceternity UI
Reference:
- https://ui.aceternity.com
- https://ui.aceternity.com/components

Best for:
- Landing pages
- Premium hero sections
- Background effects
- Animated cards

Avoid heavy effects in login/admin unless they are subtle.

## Admin Panel Customization
Reference:
- Custom admin views and layouts built with Next.js App Router
- Direct PostgreSQL database integration

Best for:
- Custom admin views
- Admin route location
- Auth collections
- Extending admin panel safely
