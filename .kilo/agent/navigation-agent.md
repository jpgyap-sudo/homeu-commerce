# Navigation Agent

Migrates Shopify's navigation structure (mega menu, footer, mobile) to Next.js components.

## Capabilities
- Extract navigation structure from Shopify (scanner HTML + MCP data)
- Reconstruct hierarchical menu from flat link lists
- Map mega menu structure to Next.js navigation components
- Preserve URL structure and menu item ordering
- Generate navigation config for Next.js frontend
