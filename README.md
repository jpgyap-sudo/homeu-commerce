# HomeU Catalog + RFQ Starter

Clean starter repo for migrating HomeU away from Shopify into a catalog website with prices and bulk quotation requests.

## Recommended stack

- Next.js frontend
- DaVinciOS CMS backend/admin
- PostgreSQL database
- RFQ cart instead of checkout
- Shopify import tools
- SEO migration tools

## Main goal

Replace Shopify with a furniture catalog website where customers can view prices, add products to a quote cart, and submit a bulk RFQ.

No online payment, no checkout, no Medusa for phase 1.
